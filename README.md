# Drug Allergen Checker

<img src="screenshots/home.png" width="600">

This web tool helps you find medications that dont contain allergens. 

This data is provided by the FDA.

https://drugallergenchecker.onrender.com/

## Features

- Search any medication by name
- Filter by common allergens (gluten, red dye, etc.)
- Compare safe vs. unsafe formulations
- View top adverse events from FDA reports
